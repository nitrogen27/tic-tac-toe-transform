# Исправление ошибок памяти GPU

## Проблема
Ошибка: `Error: Invalid TF_Status: 8 Message: failed to allocate memory`

## Внесенные изменения

### 1. Оптимизация параметров модели
- **dModel**: 256 → **192** (уменьшено для экономии памяти)
- **numLayers**: 6 → **5** (уменьшено)
- **heads**: 8 → **6** (уменьшено)
- **batchSize**: 2048 → **1024** (уменьшено для основного обучения)
- **batchSize для дообучения**: 512 → **256** (уменьшено)

### 2. Обработка ошибок памяти в train_ttt3_transformer_service.mjs
- Добавлена обработка ошибок при создании тензоров
- Добавлена обработка ошибок при вызове `model.fit()`
- При ошибке памяти батч пропускается, обучение продолжается
- Добавлена очистка памяти GPU через `tf.engine().startScope()/endScope()`

### 3. Обработка ошибок памяти в service.mjs (trainOnGames)
- Добавлена автоматическая обработка ошибок памяти
- Автоматическое уменьшение batch size при ошибке (деление пополам)
- До 3 попыток с уменьшенным batch size
- Минимальный batch size: 32

## Результаты
- ✅ Тесты полного цикла обучения проходят успешно
- ✅ Ошибки памяти обрабатываются gracefully
- ✅ Обучение продолжается даже при ошибках памяти отдельных батчей

## Рекомендации
1. Если ошибки памяти продолжаются, можно дополнительно уменьшить:
   - batchSize до 512
   - dModel до 128
   - numLayers до 4

2. Для мониторинга использования памяти GPU:
   ```bash
   docker exec tic-tac-toe-server bash -c "nvidia-smi --query-gpu=memory.used,memory.total --format=csv,noheader"
   ```
